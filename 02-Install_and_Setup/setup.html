How we use TypeScript

File has to end in a .ts or .tsx if you want JSX syntax for React.

TypeScript is always watching, even in our editor. Don't need to have it run anything before it can tell stuff is wrong. CTD.

Running the code in the integrated terminal with the tsc NAME_OF_FILE.ts will create a compiled version of the same file in JavaScript
